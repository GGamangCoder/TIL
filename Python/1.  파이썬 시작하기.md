### 1.3 리스트  
```py
>>> Fruit = ['apple', 'banana', 'grape', 'orange']
>>> len(Fruit)  # 리스트의 크기(n)
4
>>> Fruit[2]  # 리스트에서 넘버링은 0 ~ (n-1), [2]는 실제로는 3번째 인수이므로 다음과 같다.
grape
```
