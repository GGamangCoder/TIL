## 4. 데이터 타입  

### 4.1 자료형
* type() 함수  
: 자료형 파악
```py
# 예제
type(6)             # 출력; <type 'int'>, 정수
type('A')           # 출력; <type 'str'>, 문자열
```

* 숫자
- int, float, complex, ...

* 시퀀스  
- str, list, tuple

* 매핑 - dict(딕셔너리)
* 불(bool) - 참/거짓
* 세트(set) - 집합; **중복되는 원소를 갖지 않는다.**
