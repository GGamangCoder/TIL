# HTTP
## Admin Site

- admin 계정 생성; `$ python manage.py createsuperuser`
- 모델 record 보기 위해 admin.py 에 등록  
```py
# articles/admin.py

from django.contrib import admin
from .models import Article         # 추가

# Register your models here.
admin.site.register(Article)        # 추가
```
