## Design Pattern in Django

### MTV 패턴  
```
cf) MVC; Model-View-Controller
- Model; 데이터와 관련된 로직을 관리
- View; 레이아웃과 화면 처리
- Controller; 명령을 model과 view부분으로 연결
```

- Model  
    - 데이터 관련 로직
    - 응용 프로그램의 데이터 구조를 정의하고 DB 기록 관리  

- Template  
    - 레이아웃과 화면 처리
    - 사용자 인터페이스 구조와 레이아웃 정의

- View  
    - Model & Template과 관련된 로직 처리하여 응답 반환
    - 클라이언트 요청에 처리를 **분기**
